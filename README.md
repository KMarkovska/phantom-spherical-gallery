# Phantom Spherical Gallery

An interactive portfolio gallery inspired by the spatial presentation of Phantom Studios.

## Features

- Three.js gallery arranged around the inside of a sphere
- Eased click-and-drag, wheel, and keyboard navigation
- GSAP hover and project-page transitions
- Responsive desktop and mobile layouts
- Static Netlify deployment configuration

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
```

The production output is generated in `dist/`.
